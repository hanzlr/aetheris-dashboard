# Security Policy

## Supported Versions

Aetheris is a continuously deployed Discord bot — only the latest version running on the `main` branch is supported with security updates. Older tagged releases are not actively maintained.

| Version          | Supported          |
| ---------------- | ------------------- |
| Latest (`main`)  | :white_check_mark: |
| Older releases   | :x:                 |

## Reporting a Vulnerability

If you discover a security vulnerability in Aetheris (e.g. authentication bypass, exposed credentials, data leakage between servers, injection vulnerabilities), please report it responsibly:

- **Email**: raihan.ramandha@nexalab.my.id
- **Do not** open a public GitHub issue for security vulnerabilities — this could expose the issue before it's fixed.

When reporting, please include:
- A description of the vulnerability and its potential impact
- Steps to reproduce the issue
- Any relevant logs, screenshots, or proof-of-concept code

### What to expect

- You'll receive an acknowledgment of your report within **3 business days**.
- We'll investigate and provide an update on the status within **7 days**.
- If the vulnerability is confirmed, we'll work on a fix and aim to deploy it as soon as possible. You'll be credited for the discovery (unless you prefer to stay anonymous).
- If the report is declined (e.g. not reproducible, out of scope), we'll explain the reasoning.

Thank you for helping keep Aetheris and its users safe.
